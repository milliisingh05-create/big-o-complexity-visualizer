# 🚀 Big O Complexity Visualizer

An interactive web application that helps students understand **Big O Time Complexity** through visual graphs, simulations, sorting animations, and a quick-reference cheat sheet.

The goal of this project is to make algorithm complexity intuitive and engaging rather than something students memorize mechanically.

---

## 🌟 Features

### 📈 Complexity Comparison Graph
- Compare common time complexities:
  - O(1)
  - O(log n)
  - O(n)
  - O(n log n)
  - O(n²)
  - O(2ⁿ)
- Interactive graph visualization.
- Toggle complexity curves on and off.

### 🎛️ Input Size Simulator
- Adjust input size using a slider.
- Observe how operation counts grow in real time.
- Understand algorithm scalability visually.

### 📖 Complexity Explainer
- Beginner-friendly explanations.
- Real-world analogies.
- Examples of algorithms for each complexity class.

### 🔄 Sorting Visualizer
- Bubble Sort
- Merge Sort
- Quick Sort
- Animated execution.
- Displays Time Complexity and Space Complexity.

### 📚 Complexity Cheat Sheet
- Searchable algorithm list.
- Time complexity reference.
- Space complexity reference.
- Quick revision before coding interviews and exams.
